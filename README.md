Chiptune in ruby in 5 minutes
=============================
Lightning talk from [Rubyfuza](http://rubyfuza.org) 2012


### Set up the code (only works on Ubuntu Oneiric)

    bundle install
    rake setup

### See the slides

    rake slides

### Look at the lame talk code

    vim talk.rb

### Play the tunes!

    rake demo
